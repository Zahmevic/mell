Week 03. Readings

JavaScript: Novice to Ninja

Chapter 5. "Objects"

In this chapter I learned that objects are a collection of key-value placed inside brackets "{}", also I learned that objects have properties, and this properties can be any JavaScript value. I read about JSON, math and date objects and how can I use them in JavaScript

Chapter 6. "The Document Object Model"

In this Chapter I learned about the Document Object Model (DOM), and the chapter talked about document.getElementById, ByClassName, ByTagName, and the querySelector. The Book says that this can be used to access elements on the page. I also read about the parentNode(), previousSibling(), nextSibling(), childNodes() and children() methods, this are used to navigate around the DOM tree. To access to the element attributes we can use the getAttribute() method, and to update the attibute, we can use setAttribute() method.

Chapter 7. "Events"

In this chapter I learned that events occur when a user interacts with a web page. An event is attached to and element, and then invokes a callback function. When this occur, the event object pass as an argument, and it contains lots of properties and methods. There are many types of events.
To remove and event we can use the removeEventListener method.
